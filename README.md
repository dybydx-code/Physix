# ⚛️ Physix

A simple and interactive physics calculator built with HTML, CSS, and JavaScript.

## 🚀 Features
- Select physics formulas
- Input variables dynamically
- Instant calculation results
- Clean and simple UI

## 🧠 Example formulas
- v = u + at
- s = ut + ½at²
- F = ma

## 🛠️ Built With
- HTML
- CSS
- JavaScript
